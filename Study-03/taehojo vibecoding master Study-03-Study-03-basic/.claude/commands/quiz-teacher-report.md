# 학생 개인 상세 리포트

특정 학생의 전체 퀴즈 기록을 분석하여 종합 성적표를 출력합니다.

## 인수 파싱

`$ARGUMENTS`를 확인하세요.

- **학생명 지정**: 해당 학생의 리포트를 출력합니다.
- **인수 없음**: 사용법 안내를 출력합니다.

```
사용법: /quiz-teacher-report [학생명]
예시:
  /quiz-teacher-report 김민준
  /quiz-teacher-report 이수아
```

시작 전 아래 형식으로 출력하세요:
```
리포트 대상: {학생명} | 기준 일시: YYYY-MM-DD
```

---

## STEP 1 — 데이터 로드

`students/{학생명}.json`을 읽으세요.

파일이 없으면:
```
❌ 학생 데이터 없음: {학생명}
조치: /quiz-teacher-register {학생명} 으로 먼저 등록하세요.
```

점수 기록이 0개면:
```
⚠️ {학생명} 학생의 점수 기록이 없습니다.
조치: /quiz-teacher-register {학생명} [점수] [정답/총문제] 로 점수를 입력하세요.
```

---

## STEP 2 — 기본 정보 및 핵심 지표

```
========================================
  [{학생명}] 개인 성적 리포트
  등록일: YYYY-MM-DD | 기준일: YYYY-MM-DD
========================================

  핵심 지표
  ┌──────────────────┬───────────────────────────┐
  │ 항목             │ 값                        │
  ├──────────────────┼───────────────────────────┤
  │ 총 플레이 횟수   │ N회                       │
  │ 최고 점수        │ N점 (YYYY-MM-DD)          │
  │ 평균 점수        │ N.N점                     │
  │ 최고 정답률      │ N.N% (YYYY-MM-DD)         │
  │ 평균 정답률      │ N.N%                      │
  │ 최장 연속 정답   │ N회                       │
  │ 평균 응답 시간   │ N.Ns                      │
  │ 선호 모드        │ {가장 많이 플레이한 모드} │
  └──────────────────┴───────────────────────────┘
```

## STEP 3 — 점수 추이 (날짜별)

모든 게임 기록을 날짜순으로 나열하세요.

```
  점수 추이

  회차  날짜        총점   정답률  연속  모드
  ────────────────────────────────────────────────
   1회  YYYY-MM-DD  N점    N.N%   N회   {모드}
   2회  YYYY-MM-DD  N점    N.N%   N회   {모드}
   ...

  추이 분석
    첫 플레이  : N점 (YYYY-MM-DD)
    최근 플레이: N점 (YYYY-MM-DD)
    변화량     : +N점 / -N점 (N% 향상 / 하락)
    최고 점수  : N점 (N회차, YYYY-MM-DD)
    최저 점수  : N점 (N회차, YYYY-MM-DD)
```

플레이 횟수가 3회 이상이면 ASCII 추이 그래프를 추가하세요:

```
  점수 추이 그래프 (최대 10회)

  N점 │
     │             ★
     │      ★     │
     │   ★  │  ★  │
     │★  │  │  │  │
      ─────────────────
      1  2  3  4  5회차
```

## STEP 4 — 카테고리별 성적

모든 게임의 카테고리별 누적 정답을 집계하고 등급을 부여하세요.

```
  카테고리별 성적

  카테고리      정답/총문제  정답률   등급   시각화 (20칸)
  ───────────────────────────────────────────────────────
  한국사        N / N       N.N%    [등급]  [████████████░░░░░░░░]
  세계지리      N / N       N.N%    [등급]  [████████░░░░░░░░░░░░]
  과학          N / N       N.N%    [등급]  [████████████████████]
  예술과 문화   N / N       N.N%    [등급]  [████░░░░░░░░░░░░░░░░]
```

등급 기준:
- **S**: 90% 이상
- **A**: 80% 이상
- **B**: 70% 이상
- **C**: 60% 이상
- **D**: 50% 이상
- **F**: 50% 미만

```
  선생님 메모란
  (이 항목은 선생님이 직접 추가할 수 있는 자유 기입란입니다)
  ─────────────────────────────────────────────────────
  (비어 있음 — /quiz-teacher-register {학생명} 실행 후 JSON 파일에 "memo" 필드로 추가 가능)
```

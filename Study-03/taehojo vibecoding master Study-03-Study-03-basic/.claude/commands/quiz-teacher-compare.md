# 학생 간 성적 비교

지정된 2명 이상의 학생 데이터를 읽어 항목별로 나란히 비교합니다.

## 인수 파싱

`$ARGUMENTS`를 공백으로 분리하여 학생명 목록으로 사용하세요.

- **2명 이상 지정**: 해당 학생들을 비교합니다.
- **1명만 지정**: 사용법 안내를 출력하고 종료합니다.
- **인수 없음**: `students/` 디렉토리의 전체 학생을 비교합니다 (최대 6명, 초과 시 안내).

사용법 안내:
```
사용법: /quiz-teacher-compare [학생명1] [학생명2] [학생명3 ...]
예시:
  /quiz-teacher-compare 김민준 이수아
  /quiz-teacher-compare 김민준 이수아 박지호
  /quiz-teacher-compare          ← 전체 학생 비교
```

시작 전 비교 대상을 표시하세요:
```
비교 대상: [이름1] vs [이름2] vs ... (N명)
```

---

## STEP 1 — 데이터 로드 및 검증

지정된 학생의 `students/{학생명}.json` 파일을 각각 읽으세요.

파일이 없는 학생이 있으면:
```
⚠️ 파일 없음: {학생명} → students/{학생명}.json 이 존재하지 않습니다.
```
해당 학생을 제외하고 나머지 학생으로 계속 진행합니다. 1명만 남으면 비교 불가로 종료합니다.

점수 데이터가 없는 학생은 비교 항목을 `-`로 표시합니다.

---

## STEP 2 — 기본 수치 비교

각 학생의 전체 게임 기록을 집계하여 나란히 출력하세요.

```
========================================
  학생 성적 비교
  기준 일시: YYYY-MM-DD
========================================

  항목            │ [이름1]  │ [이름2]  │ [이름3]
  ────────────────┼──────────┼──────────┼──────────
  최고 점수       │    N점   │    N점   │    N점
  평균 점수       │   N.N점  │   N.N점  │   N.N점
  평균 정답률     │   N.N%   │   N.N%   │   N.N%
  최장 연속 정답  │    N회   │    N회   │    N회
  총 플레이 횟수  │    N회   │    N회   │    N회
  최근 플레이     │ YYYY-MM  │ YYYY-MM  │ YYYY-MM
```

각 행에서 가장 높은 값을 가진 학생의 셀에 `★` 표시를 추가하세요 (동점 시 모두 표시).

---

## STEP 3 — 카테고리별 정답률 비교

각 학생의 누적 카테고리별 정답률을 계산하여 비교하세요.

```
  카테고리별 정답률 비교

  카테고리      │ [이름1]   │ [이름2]   │ [이름3]
  ─────────────┼───────────┼───────────┼───────────
  한국사        │  N.N%     │  N.N%     │  N.N%
  세계지리      │  N.N%     │  N.N%     │  N.N%
  과학          │  N.N%     │  N.N%     │  N.N%
  예술과 문화   │  N.N%     │  N.N%     │  N.N%
```

카테고리별 1위 학생에 `★` 표시.
카테고리 데이터가 없으면 `-`로 표시하세요.

---

## STEP 4 — 강점 / 약점 요약

각 학생의 카테고리 정답률 기반으로 강점과 약점을 한 줄 요약하세요.

```
  강점 / 약점 요약

  [이름1]
    강점: {가장 높은 정답률 카테고리} (N.N%)
    약점: {가장 낮은 정답률 카테고리} (N.N%)

  [이름2]
    강점: {가장 높은 정답률 카테고리} (N.N%)
    약점: {가장 낮은 정답률 카테고리} (N.N%)
```

---

## STEP 5 — 점수 추이 비교

각 학생의 날짜별 최고 점수를 나열하여 성장 흐름을 비교하세요.
(플레이 기록이 2회 이상인 학생만 포함)

```
  점수 추이 (날짜별 최고 점수)

  날짜        │ [이름1] │ [이름2] │ [이름3]
  ────────────┼─────────┼─────────┼─────────
  YYYY-MM-DD  │   N점   │    -    │   N점
  YYYY-MM-DD  │    -    │   N점   │   N점
  ...

  성장률 (첫 플레이 대비 최근 플레이):
    [이름1]: N점 → N점 (+N점 / -N점)
    [이름2]: N점 → N점 (+N점 / -N점)
```

---

## STEP 6 — 비교 결론

```
========================================
  비교 결론
========================================

  종합 우수: {이름} (최고 점수 + 정답률 + 연속 정답 합산 기준)

  항목별 1위 집계
    최고 점수    : {이름} (N점)
    평균 정답률  : {이름} (N.N%)
    최장 연속    : {이름} (N회)
    카테고리 1위 수: {이름} (N개 카테고리)

  추천 보충 학습
    [이름1]: {약점 카테고리} 집중 권장
    [이름2]: {약점 카테고리} 집중 권장
```

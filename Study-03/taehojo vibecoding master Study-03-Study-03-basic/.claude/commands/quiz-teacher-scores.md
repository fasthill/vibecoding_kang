# 전체 학생 점수 현황 및 랭킹

`students/` 디렉토리의 모든 학생 데이터를 읽어 점수 현황과 순위표를 출력합니다.

## 인수 파싱

`$ARGUMENTS`를 확인하세요.

| 인수 | 동작 |
|------|------|
| (없음) | 최고 점수 기준 랭킹 출력 |
| `accuracy` | 평균 정답률 기준 랭킹 |
| `plays` | 총 플레이 횟수 기준 랭킹 |
| `streak` | 최장 연속 정답 기준 랭킹 |
| `recent` | 가장 최근에 플레이한 순서로 정렬 |
| `avg` | 평균 점수 기준 랭킹 |

시작 전 아래 형식으로 모드를 표시하세요:
```
정렬 기준: [최고 점수 / 평균 정답률 / 총 플레이 횟수 / 최장 연속 / 최근 플레이 / 평균 점수]
```

---

## STEP 1 — 학생 데이터 수집

`students/` 디렉토리의 모든 `.json` 파일을 읽으세요.

**디렉토리가 없거나 파일이 0개인 경우:**
```
❌ 등록된 학생이 없습니다.
조치: /quiz-teacher-register [학생명] 으로 학생을 먼저 등록하세요.
```

데이터를 수집한 뒤 아래 형식으로 기본 현황을 출력하세요:
```
--- 데이터 수집 완료 ---
등록 학생 수   : N명
총 게임 기록 수 : N회
점수 없는 학생  : N명 (이름 목록)
```

---

## STEP 2 — 통계 집계

각 학생에 대해 다음을 계산하세요.

| 항목 | 계산 방식 |
|------|-----------|
| 최고 점수 | `scores` 배열 중 `totalScore` 최댓값 |
| 평균 점수 | `totalScore` 합계 ÷ 게임 수 (소수점 1자리) |
| 평균 정답률 | `accuracy` 합계 ÷ 게임 수 (소수점 1자리) |
| 최장 연속 정답 | `longestStreak` 최댓값 |
| 총 플레이 횟수 | `scores` 배열 길이 |
| 최근 플레이 | `date` 기준 가장 최근 값 |

점수 데이터가 없는 학생은 통계를 `-`로 표시하고 랭킹에서 하단에 배치하세요.

---

## STEP 3 — 랭킹 출력

지정된 기준으로 내림차순 정렬하여 출력하세요.

```
========================================
  학생 점수 현황 — [정렬 기준] 기준
  기준 일시: YYYY-MM-DD
========================================

  순위  학생명     최고점  평균점  정답률  연속  플레이  최근 플레이
  ────────────────────────────────────────────────────────────────
   1위  (이름)     NNN점   NN.N점  NN.N%  N회   N회     YYYY-MM-DD
   2위  (이름)     NNN점   NN.N점  NN.N%  N회   N회     YYYY-MM-DD
   3위  (이름)     NNN점   NN.N점  NN.N%  N회   N회     YYYY-MM-DD
   ...
  (미기록) (이름)    -       -       -      -     0회     -
```

1~3위에는 메달 표시 추가 (1위: 🥇, 2위: 🥈, 3위: 🥉)

---

## STEP 4 — 클래스 전체 요약

```
========================================
  클래스 전체 요약
========================================

  등록 학생 수   : N명 (기록 있는 학생: N명)
  총 게임 횟수   : N회
  클래스 최고 점수: N점 ({이름}, YYYY-MM-DD)
  클래스 평균 점수: N.N점
  클래스 평균 정답률: N.N%
  클래스 최장 연속: N회 ({이름})

  모드별 플레이 분포
    전체 도전 모드   : N회 (N%)
    카테고리별 모드  : N회 (N%)
    스피드 퀴즈 모드 : N회 (N%)
```

---

## STEP 5 — 주목 학생 하이라이트

```
  주목 학생
  ┌──────────────────┬─────────────────────────────┐
  │ 항목             │ 학생 (값)                   │
  ├──────────────────┼─────────────────────────────┤
  │ 최고 점수        │ {이름} — N점                │
  │ 최고 정답률      │ {이름} — N.N%               │
  │ 최장 연속 정답   │ {이름} — N회                │
  │ 가장 많이 플레이 │ {이름} — N회                │
  │ 가장 빠른 성장   │ {이름} — 첫 플레이 대비 +N점│
  └──────────────────┴─────────────────────────────┘
```

가장 빠른 성장: 가장 최근 점수 - 첫 번째 점수가 가장 높은 학생
(데이터가 2회 이상인 학생 중에서만 계산)

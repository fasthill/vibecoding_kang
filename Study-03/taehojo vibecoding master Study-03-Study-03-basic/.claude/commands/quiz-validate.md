# 퀴즈 최상급 표현 검증

questions.js 파일을 읽고, 아래 조건에 따라 최상급 표현이 포함된 문제(question 필드)를 찾아 검증해 주세요.

## 검증 범위

$ARGUMENTS 값을 확인하세요.

- **카테고리가 지정된 경우** (`$ARGUMENTS`가 비어 있지 않으면): 해당 카테고리(category 필드)와 일치하는 문제만 검증하세요.
- **카테고리가 지정되지 않은 경우** (`$ARGUMENTS`가 비어 있으면): 전체 문제를 검증하세요.

검증 시작 전 첫 줄에 아래 형식으로 범위를 표시하세요.

```
검증 범위: [전체] 또는 [카테고리: (카테고리명)]
```

## 검색 대상 표현

아래 표현이 포함된 문제를 찾고, 각 표현에 어떤 기준을 명시해야 하는지 함께 안내하세요.

| 표현 | 명시해야 할 기준 예시 |
|------|----------------------|
| 가장 | 측정 기준 (예: 면적, 높이, 인구 수 등) |
| 최초 | 기준 범위 (예: 한국 최초, 세계 최초) 및 시점 |
| 최대 | 측정 단위 및 기준 (예: 용량 기준, 면적 기준) |
| 최고 | 측정 기준 (예: 해발고도 기준, 온도 기준) |
| 최소 | 측정 단위 및 기준 |
| 최장 | 측정 기준 (예: 길이 기준, 기간 기준) |
| 처음 | 기준 범위 및 시점 |
| 제일 | 측정 기준 |

## 출력 형식

각 항목을 아래 형식으로 표시하세요.

```
[문제 ID] 카테고리 / 난이도
문제: (문제 내용)
포함된 표현: (예: 가장, 최대)
검증 필요 여부: ✅ 기준 명시됨 / ⚠️ 기준 불명확
권장 수정 방향: (기준 불명확인 경우, 어떤 기준을 추가해야 하는지 구체적으로 안내)
```

## 검증 기준

- ✅ 기준 명시됨: 문제에 측정 기준이 함께 적혀 있는 경우 (예: "면적 기준으로", "해발고도 기준으로")
- ⚠️ 기준 불명확: 측정 기준 없이 최상급 표현만 사용된 경우

마지막에 전체 요약을 출력하세요.

```
--- 검증 요약 ---
검증 범위: [전체] 또는 [카테고리: (카테고리명)]
검증 대상 문제 수: N개
최상급 표현 포함 문제 수: N개
  ✅ 기준 명시됨: N개
  ⚠️ 기준 불명확: N개
```
